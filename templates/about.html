Food Detector

Web App for Food Detection using YOLOv5

5 datasets are aggregated for use: Open Images V6 - Food, School Lunch Dataset, Vietnamese Food, MAFood-121, Food-101

93,748 training images, 26,825 evaluating images, up to 180 classes

4 versions of YOLOv5: (s, m, l, x)

Integrated with Edamam API for nutrition analysis based on food names

Instructions for use

Upload A File (Image/ Video) or Pasting A URL (Image/ Youtube URL)

Provide some additional features to increase interactivity with users:

Confidence scores and IOU threshold are adjustable

Choose 1 of 4 YOLOv5 versions to use

Label enhancement to further predict "Other Food" class using EfficientNet-B4

Ensemble 4 YOLOv5 models and aggregated by "weighted box fusion" method

Provide 2 types of Nutrition Statistics Chart below an image result (by food and by nutrition)
